      New user settings have arrived - try them out now

      less than 1 minute read

      Summary: Announcing new user features at Dbljump including profile picture uploads, non-binary gender settings, auto-redirects after logging in, and login email changes.

      Have you registered as a Dbljump editor yet? If you have, you might have noticed a few missing features at our user settings site id.dbljump.com.

      Good news - we just added a raft of features that you probably expected to be there already!

      You can now…

      • Upload, change, and remove your profile picture - feel like you’re in the future yet?
      • Set your location and gender - choose binary options or type whatever you want
      • Be taken back to the page you clicked ‘Log in’ from - automatically as if by magic!
      • Set a new login email - you’ll be asked to confirm it before you can start using it

      If you run into any bugs, problems, or just want to let us know what you think, please tweet us or talk to us via our Facebook page.

      Here ends this short announcement. Want to join our mission to organise the world’s gaming knowledge? Sign up here.
